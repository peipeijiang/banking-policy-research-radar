<details>
<summary><strong>英文摘要</strong></summary>

We introduce the mnorm package, which allows one to calculate conditional multivariate normal densities and probabilities and to differentiate them with respect to various parameters including covariances and integration limits. The package also supports parallel (multi-core) computing, handles non-normal marginals via the Gaussian copula, and provides fast routines for the calculation of bivariate and trivariate normal probabilities. The package is of special interest for the implementation of the maximum-likelihood estimators in econometric models.
