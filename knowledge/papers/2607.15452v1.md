<summary><strong>中文摘要</strong></summary>

关于无限博弈中纳什均衡存在性的研究已演变为一个由技术性前提条件和反例拼凑而成的领域。本文通过修正基于可数可加性的主流混合策略模型，提出了一种均衡理论的统一方案。一个博弈由非空参与者集合、每个参与者的非空行动集以及有界冯·诺伊曼-摩根斯坦效用函数定义。本文证明，每个此类博弈在有限可加混合策略下均存在纳什均衡。此外，任何此类博弈的均衡对应被证明是非空的、紧值的且上半连续的，且作为有限逼近极限的均衡同样具有这些性质。本文发展的技术表明，长期被视为棘手的无限博弈现在可直接进行均衡分析。

<summary><strong>英文摘要</strong></summary>

Research on Nash equilibrium existence for infinite games has grown into a patchwork of technical preconditions and counterexamples. This paper presents a unified program in equilibrium theory by revising the predominant model of mixed strategies based on countable additivity. A game is specified by a nonempty set of players and, for each player, a nonempty action set and a bounded von Neumann-Morgenstern utility function. Every such game is shown to admit a Nash equilibrium in finitely additive mixed strategies. In addition, the equilibrium correspondence for any such game is shown to be nonempty, compact-valued, and upper hemicontinuous, and the same is true for equilibria obtained as limits of finite approximations. Techniques developed in this paper show that infinite games long treated as intractable become amenable to direct equilibrium analysis.
